# 🏎️ FP Selection - Luxury Car Showcase



## ✨ About The Project

**FP Selection** is an interactive luxury car showcase, built with HTML5, CSS3, and pure JavaScript. The project delivers an immersive visual experience with smooth transitions and modern design, highlighting the most iconic vehicles in the automotive world.

### 🚗 Featured Cars

| Car | Category | Highlight |
|-----|----------|-----------|
| Porsche 911 Turbo | Sports Car | Brand New In Stock |
| Ferrari Portofino | Grand Tourer | Only In Stock |
| Lamborghini Aventador | Supercar | Best Selling |

## 🎯 Features

- ✅ **Interactive Carousel** - Smooth navigation between vehicles
- ✅ **Visual Indicators** - Dots and current slide number
- ✅ **CSS Animations** - Fluid and elegant transitions
- ✅ **Modern Interface** - Gradients, hover effects, and premium typography

## 🛠️ Technologies Used

| Technology | Description |
|------------|-------------|
| HTML5 | Semantic project structure |
| CSS3 | Styling, animations, and responsive layout |
| JavaScript | Carousel logic and interactivity |
| Google Fonts | Poppins and League Gothic for typography |
